## Record

一个用于局域网音频传输的程序
A LAN audio transmission for desktop apps

## 使用

1. 下载程序
2. 双方机器分别运行程序
3. 在音频输出机器的程序上输入输入机器的IP地址
4. 点击建立连接

## 开发（Code）

### 安装依赖（Install）

```bash
pnpm install
```

### 开始开发（Starting Development）

```bash
pnpm start
```

### 打包并部署（Packaging for Production）

```bash
pnpm package
```
